# 微信推文封面提取工具（wechat-cover-tool）

输入一篇微信推文链接（或粘贴网页源代码），自动提取它的两张头图：

- **`cdn_url_1_1`** → 正方形封面（1:1）
- **`cdn_url_235_100`** → 横幅封面（235:100）

支持批量链接、自动命名、一键打包下载。

---

## 三种用法

### 1. 在线版（GitHub Pages，本仓库）
打开 **https://moone7.github.io/wechat-cover-tool/** 即可使用。

- **粘贴源代码模式（在线 100% 可用）**：右键推文 → 查看网页源代码 → 全选复制 → 粘贴 → 解析。图片经 CORS 代理读取以绕过微信防盗链。
- **输入链接模式**：每行一个链接 → 提取。同样经 CORS 代理抓取，微信偶尔会临时封锁代理 IP，失败时用「粘贴源代码」模式。
- 在线版图片显示/下载依赖公共 CORS 代理，链接与内容会经过第三方服务器；敏感/客户内容建议用下面的本机版。

### 2. 本机 Python 版（推荐，数据不出本机，无需代理）
```bash
python wechat_cover_tool.py
# 浏览器打开 http://localhost:8765
```
- 服务端用真实浏览器请求头抓取文章、解析 `var msgList`，下载时经本机代理绕开防盗链。
- 双击 `启动封面提取工具.bat` 可一键启动并自动打开浏览器。
- 链接模式、批量、自动命名、打包 zip 全部可用，且不需要任何第三方代理。

### 3. 纯前端静态文件
`index.html` 本身即可作为静态站点部署到任意支持 Pages 的平台（GitHub Pages / Vercel / Netlify 等）。

---

## 文件说明
- `index.html` — 纯前端在线版（可托管 GitHub Pages）
- `wechat_cover_tool.py` — 本机 Python 服务端版（零依赖，端口 8765）
- `启动封面提取工具.bat` — Windows 一键启动器
- `README.md` — 本说明

## 隐私说明
- 本机 Python 版：仅抓取你输入的公开推文，数据不出本机。
- 在线版：链接/图片经公共 CORS 代理中转，请勿用于涉密内容。
